## 🎮 立即行动指南

```bash

python3 start_web_server.py
```

### **步骤2：获取访问地址**
```bash
# 查看服务器IP
ip -4 addr show | grep inet | grep -v 127.0.0.1

# 示例输出：
# inet 192.168.1.100/24 brd 192.168.1.255 scope global eth0
```

## ❓ 常见问题解答

### Q1：手机无法访问？
- **检查网络**：确保手机和服务器同一网络
- **检查防火墙**：`sudo ufw allow 8789`
- **检查IP**：确认使用正确的服务器IP

### Q2：文件更新不及时？
- 每天06:55自动更新
- 手动更新：复制文件到Web目录
- 刷新浏览器缓存

### Q3：想在公网访问？
- 使用ngrok：`ngrok http 8789`
- 获取临时公网地址
- 分享给其他设备

### Q4：文件格式乱码？
- 确保.md文件UTF-8编码
- 浏览器设置UTF-8
- 使用Markdown阅读器应用
