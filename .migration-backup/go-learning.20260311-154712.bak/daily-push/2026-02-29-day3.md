### 💻 核心代码对比
```go
// Go (简洁统一)
for i := 0; i < 5; i++ {
    fmt.Println(i)
}

// while样式的for
count := 0
for count < 10 {
    count++
}

// switch (不需要break!)
switch grade {
case "A":
    fmt.Println("优秀")
case "B":
    fmt.Println("良好")
default:
    fmt.Println("其他")
}
```

```java
// Java (多种选择)
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

while (count < 10) {
    count++;
}

switch (grade) {
    case "A":
        System.out.println("优秀");
        break;  // 必须break!
    case "B":
        System.out.println("良好");
        break;
    default:
        System.out.println("其他");
}
```

## 💡 给Java开发者的快速提示

### **Go控制流关键特点**
1. **简洁性**: if语句没有括号，switch不需要break
2. **统一性**: 只有`for`循环，没有while/do-while
3. **安全性**: switch避免"贯穿"bug
4. **灵活性**: if条件中可以包含声明语句

### **今天只要记住**
- Go `for` = Java的for+while
- Go `switch` = Java switch + 自动break
- `range` 是Go的遍历神器

**阳光提醒**: Go设计者删除了多余的语法结构，让语言更一致和安全。今天重点:适应`for`的多种用法！
