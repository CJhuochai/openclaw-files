## 🔍 Java vs Go 快速对比

```java
// Java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello");
    }
}
```

```go
// Go
package main
import "fmt"

func main() {
    fmt.Println("Hello")
}
```

**关键差异**:
- Go没有"类"，直接写`main()`函数
- `fmt.Println()`比`System.out.println()`简洁
- 无需"public static"等修饰符
