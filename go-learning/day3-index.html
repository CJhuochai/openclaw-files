🎯 今日学习重点

作为Java开发者，你今天要了解Go如何简化控制流语法，并逐步适应这种简洁的设计。

🔹 Go控制流特点

简洁统一:

  • if语句没有括号
  • 只有for循环（替代while）
  • switch不需要break
  • range是唯一遍历方式

🔸 Java开发者注意事项

思维转换:

  • 忘记while关键字
  • 不需要写break
  • 习惯if的简洁写法
  • 用for代替所有循环

💻 代码对比示例

// Go循环 for i := 0; i < 5; i++ { fmt.Println(i) } // 作为while使用 count := 0 for count < 10 { count++ } // switch (安全，自动break) switch grade { case "A": fmt.Println("优秀") case "B": fmt.Println("良好") }
// Java循环 for (int i = 0; i < 5; i++) { System.out.println(i); } // while循环 int count = 0; while (count < 10) { count++; } // switch (需要break!) switch (grade) { case "A": System.out.println("优秀"); break; // 必须break! case "B": System.out.println("良好"); break; }
