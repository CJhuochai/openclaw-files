学习重点：
- Go的if不需要括号
- for是Go唯一的循环关键字
- switch可以不带表达式

对比示例：
// Java的for循环
for(int i=0; i<10; i++){}

// Go的for循环 (三种写法)
for i:=0; i<10; i++{}  // 类C写法
for i<10 { i++ }      // 类似while
for range             // 遍历

💡 发现：Go的设计追求“少即是多”！
