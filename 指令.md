## 🔧 网络与开发工具

### SSH隧道相关

#### 指令 #001 - OpenClaw WebUI访问隧道
**添加时间：** 2026-02-26 14:37  
**添加者：** 陈健  
**指令：**
```bash
ssh -N -L 18789:127.0.0.1:18789 root@43.167.172.248
```

**指令备注：** 
通过本地终端SSH隧道访问OpenClaw WebUI

**参数说明：**
- `-N`：不执行远程命令，仅建立隧道
- `-L`：本地端口转发
- `18789:127.0.0.1:18789`：本地端口:远程主机:远程端口
- `root@43.167.172.248`：服务器地址和用户名

**使用场景：**
当需要从本地浏览器访问OpenClaw WebUI时，先在终端执行此命令建立隧道，然后在浏览器中访问 `http://localhost:18789`

**常见问题：**
1. 如果端口被占用，可以修改本地端口号
   ```bash
   ssh -N -L 18888:127.0.0.1:18789 root@43.167.172.248
   ```
2. 需要确保服务器上的OpenClaw Gateway在运行
3. 可能需要配置SSH密钥认证
