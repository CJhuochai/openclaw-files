## 📝 详细步骤指导

### 步骤1：安装Go

**方法一：官方网站下载（推荐）**
1. 访问 https://golang.org/dl/
2. 选择适合你操作系统的版本：
   - Windows: `.msi` 安装包
   - macOS: `.pkg` 安装包
   - Linux: `.tar.gz` 压缩包

**方法二：包管理器安装**
```bash
# macOS (使用Homebrew)
brew install go

# Ubuntu/Debian
sudo apt install golang-go

# CentOS/RHEL
sudo yum install golang
```

### 步骤2：验证安装

打开终端/命令提示符，输入：
```bash
# 查看Go版本
go version

# 查看Go环境
go env

# 应该看到类似输出：
# go version go1.22.0 linux/amd64
```

### 步骤3：创建第一个Go程序

1. 创建一个工作目录：
```bash
mkdir -p ~/go-learning/day1
cd ~/go-learning/day1
```

2. 创建 `hello.go` 文件：
```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, 陈健！欢迎来到Go世界！")
    fmt.Println("🎉 恭喜你迈出Go第一步！")
}
```

### 步骤4：运行程序

```bash
# 方式1：直接运行
go run hello.go

# 方式2：编译后运行
go build hello.go
./hello  # Linux/macOS
hello.exe  # Windows

# 预期输出：
# Hello, 陈健！欢迎来到Go世界！
# 🎉 恭喜你迈出Go第一步！
```

## 🔍 概念解析

### Go vs Java 环境对比

| 方面 | Java | Go | 说明 |
|------|------|-----|------|
| **运行环境** | JVM（Java虚拟机） | 直接编译为二进制 | Go程序无需额外运行时 |
| **安装包** | JDK（较大） | Go工具链（较小） | Go安装包约100MB |
| **环境变量** | JAVA_HOME | GOROOT, GOPATH | Go历史遗留，现在主要用Go Modules |
| **编译命令** | javac → java | go build → ./executable | Go编译更简单 |

### 理解GOROOT和GOPATH

**GOROOT**：Go安装目录（类似JAVA_HOME）
- 存放Go标准库、工具链
- 通常自动设置，无需手动修改

**GOPATH**：传统工作区目录（历史概念）
- 存放Go代码、第三方包、编译输出
- Go 1.11+ 推荐使用Go Modules，GOPATH逐渐淘汰

**Go Modules**：现代依赖管理（类似Maven）
- 每个项目独立的依赖管理
- 使用 `go.mod` 文件定义模块
- 已经是Go标准依赖管理方案

## 💻 代码对比学习

### Hello World: Java vs Go

**Java版本：**
```java
// Hello.java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

// 编译运行：
// javac Hello.java
// java Hello
```

**Go版本：**
```go
// hello.go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}

// 编译运行：
// go run hello.go
// 或
// go build hello.go
// ./hello
```

### 主要差异点：

1. **文件组织**
   - Java：每个public类需要一个单独文件
   - Go：一个文件可以有多个函数，包名决定编译方式

2. **入口函数**
   - Java：`public static void main(String[] args)`
   - Go：`func main()`（必须属于package main）

3. **导入语句**
   - Java：`import java.util.*;`
   - Go：`import "fmt"`（双引号）

4. **输出函数**
   - Java：`System.out.println()`
   - Go：`fmt.Println()`（首字母大写）

## ❓ 常见问题解答

### Q1：安装时遇到权限问题？
- **Windows**：以管理员身份运行安装程序
- **Linux/macOS**：使用sudo或安装在用户目录

### Q2：go命令找不到？
- 需要将Go的bin目录添加到PATH环境变量
- Windows：`C:\Go\bin`
- Linux/macOS：`/usr/local/go/bin`

### Q3：GOPATH需要设置吗？
- 对于新项目，建议使用Go Modules
- 运行 `go mod init <module名>` 创建新模块
- 无需手动设置GOPATH

### Q4：IDE无法识别Go安装？
- 重启IDE
- 检查PATH环境变量
- 确认Go安装成功（`go version`）
